# Project Instructions

This is an Event Management System built with:

- ASP.NET Core Web API
- Angular
- MySQL
- Entity Framework Core
- JWT authentication
- MVC-style architecture

Rules:

- Do not generate unrelated features.
- Keep backend structure as Controllers, Services, Repositories, DTOs, Entities, Data.
- Keep business logic out of controllers.
- Use DTOs for API requests and responses.
- Use async/await for database calls.
- Validate inputs before saving.
- Follow role-based authorization.
- Write tests for service-layer business rules.
- Explain commands before asking me to run them.
